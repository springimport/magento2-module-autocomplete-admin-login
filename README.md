# Autocomplete Admin Login for Magento 2

Enable autocomplete for admin login.

## Version Compatibility

| Magento Version | PHP Version | Status |
|-----------------|-------------|--------|
| **2.3.x - 2.4.x** | **7.2 - 8.3** | **✅ Compatible** |
| 2.2.x | 7.0 - 7.2 | ✅ Compatible |
| 2.0.x - 2.1.x | 5.5 - 7.1 | ✅ Compatible |

## Installation

Enable module:
```bash
php -f bin/magento module:enable SpringImport_AutocompleteAdminLogin
```

Disable module:
```bash
php -f bin/magento module:disable SpringImport_AutocompleteAdminLogin
```

Update system:
```bash
php -f bin/magento setup:upgrade
```
